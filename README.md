# VOD Sites Manager

A world-class web app for managing and editing VOD sites, built with React, TypeScript, and Vite. Your single source of truth for all site data.

## Features

- 🌟 **Beautiful UI**: Modern, professional design with teal/cyan color scheme
- 🔍 **Powerful Search**: Search sites by name, group, services, or IP
- ➕ **Add/Edit/Delete**: Full CRUD operations
- 💾 **Local Storage**: Data persistence across sessions
- ⌨️ **Keyboard Shortcuts**: Ctrl+N to add new, Esc to close modal
- 🔗 **Quick Links**: Direct access to IPTV, Headend, WLAN controller
- ♿ **Accessibility**: ARIA labels, keyboard navigation
- 📱 **Responsive Design**: Works perfectly on all devices
- 🎨 **Smooth Animations**: Professional micro-interactions
- 🚀 **Vercel Ready**: One-click deployment

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Keyboard Shortcuts

- **Ctrl+N**: Add new site
- **Esc**: Close modal
- **Enter**: Open site card

## Deploy to Production

Your VOD Sites Manager can be deployed to **both Vercel and Azure**! See the [DEPLOYMENT.md](./DEPLOYMENT.md) guide for complete instructions.
